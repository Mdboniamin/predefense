About Food Order App

This is a multi-user role-based food ordering system with the following features:

  • Customer Features:
    • Browse food items from different restaurants
    • Add items to cart and place orders
    • Make payments via bKash
    • Track order history and status
    • Update profile information
  • Restaurant Features:
    • Manage food items (add, edit, delete)
    • View and manage orders
    • Update order status
    • View payment details
    • Update profile information
  • Admin Features:
    • Manage all users, food items, and orders
    • Approve restaurant accounts
    • Verify bKash payments
    • Full system oversight

The system uses manual bKash payment verification for secure transactions.
